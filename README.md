### 풀이
BFS를 활용한 풀이
* 1차원 set을 활용하여 이미 해킹을 당한 서버 탐색 속도를 O(1)으로 줄임
* 남아 있는 서버의 수가 2B개 미만의 경우, 계산하지 않고 바로 Break
* Matplotlib을 활용하여 시뮬레이션에 대한 통계도 아름답게 확인 가능
